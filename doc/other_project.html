Other PIA projects

This is a list of other projects that aims to read PIA file in various way:

PiaNO a library to read and write all PIA file, written in C#, released under MIT license

unzip_pc3 a command line program that compress and decompress pc3, written in C#, without a license, public domain?

ezdxf library for DXF, but can also handle .ctb and .stb, written in Python,released under MIT license


Other unrelated but interesting CAD projects

LibreDWG is a free library to handle DWG and DXF files, written in C, released under GPLv3+ license

libdxf is a free library to handle DXF files, written in C, released under GPLv2+ license

libdxfrw is a free library to handle DXF and partially DWG files, written in C++, released under GPLv2+ license

zcad is a free CAD program, written in Pascal, released under a modified LGPLv2 license (the exception can be removed, so a fork can be LGPLv2)

LibreCAD 3 is a modular 2D CAD application, written in C++, released under GPLv3+ license
