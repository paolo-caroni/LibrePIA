Plotter Configuration version 3

PC3 is a file extension for a plotter configuration used with AutoDesk AutoCAD software.
Normally all programs send output directly to system printers, AutoCAD for Windows don't (but AutoCAD for Mac OS X does!).
The PC3 is a configuration file that use Heidi® Device Interface from AutoDesk.
Since the HDI is closed source, undocumented and compatible only under Windows S.O. I'm not very interested to understand how PC3 and PMP works.
