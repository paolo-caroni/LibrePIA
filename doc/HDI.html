HDI (Heidi® Device Interface)

HDI (Heidi® Device Interface) drivers are used to communicate with hard copy devices.

WARNING: HDI (Heidi® Device Interface) it's totally different to Heidi that stands for HDI, which also stands for HOOPS Device Interface a software API developed by Ithaca Software for video card graphics driver.
The similarity of Heidi and HDI name makes much confusion.

Heidi® Device Interface it's proprietary and totally undocumented.
I don't know if it's possible to replace it with Line Printer Remote protocol (LPD) or maybe Internet Printing Protocol(IPP).

HDI drivers fall into three categories:

-HDI File format drivers
-HDI nonsystem drivers
-HDI system printer drivers

HDI File format drivers are virtual printer can print in .pdf or other file formats without any device.
HDI nonsystem drivers should be used for older 'legacy' devices that cannot be set up through the operating system (OS).
Newer devices that can be configured through the OS should be used by the program as a system printer instead of a nonsystem printer.
